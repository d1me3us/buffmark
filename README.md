# Buffmark

Chrome extension that allow to add selected text to bookmarks and then paste it into any form field.

## Use

1. Select text on any page.
2. Right-click the selection.
3. Choose **Bookmark with Buffmark**.
4. Confirm the bookmark name and folder.
5. Focus a text input or textarea on a page, then click the created bookmark.
