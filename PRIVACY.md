# Privacy Policy

**Effective Date:** July 1, 2026

This Privacy Policy describes how Buffmark, a Chrome Extension, handles information when you use the extension.

## Information the Extension Handles

Buffmark allows you to save selected text as Chrome bookmarks and paste saved bookmark content into form fields.

To provide this functionality, the extension may handle the following information:

* Text selected by the user on a web page
* Bookmark data created by the user, such as bookmark title, content, URL, and folder location
* Extension settings, such as the last-used bookmarks folder

## How Information Is Used

The extension uses this information only to provide its core functionality:

* Creating bookmarks from selected text
* Showing context menu actions
* Saving the last-used bookmarks folder for convenience
* Allowing users to paste saved content into form fields

The extension does not use this information for advertising, analytics, profiling, or unrelated purposes.

## Data Storage

Buffmark stores bookmark data using Chrome’s built-in bookmarks functionality. Extension settings, such as the last-used bookmarks folder, may be stored using Chrome extension storage.

The extension does not operate an external server for storing user data.

## Data Sharing and Disclosure

Buffmark does not sell, rent, transfer, or share user data with third parties.

The extension does not transmit selected text, bookmark data, or extension settings to the developer or to any external service.

## Limited Use

Buffmark uses user data only for the purpose of providing and improving the extension’s single purpose: saving selected text as bookmarks and allowing saved content to be pasted into form fields.

The extension does not use or transfer user data for purposes that are unrelated to its core functionality.

## User Control and Data Deletion

Users can manage or delete bookmarks created by the extension through Chrome’s built-in bookmark manager.

Users can remove extension settings by uninstalling the extension or clearing the extension’s stored data through Chrome.

## Children's Privacy

Buffmark is not intended for children under the age of 13. The extension does not knowingly collect personally identifiable information from children under 13.

## Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated effective date. Users are advised to review this Privacy Policy periodically for changes.
